# Util Filter File
**Util Filter File** - это Java программа для анализа содержимого текстовых файлов. Она классифицирует данные на три типа: целые числа, числа с плавающей точкой и строки, создает отдельные файлы для каждого типа и выводит статистику по ним.

---

## Функционал

- Чтение нескольких текстовых файлов `.txt`.
- Классификация данных:
    - **int** - целые числа.
    - **float** - числа с плавающей точкой.
    - **string** - текстовые строки.
- Генерация выходных файлов для каждого типа данных с возможностью указания префикса.
- Вывод статистики:
  - **Краткая статистика:** количество элементов в output файлах
  - **Полная статистика:** количество, сумма, среднее, минимум, максимум (для чисел) и длина строк (для строк).

---

## Структура проекта

- `Main.java` - содержит логику обработки файлов и подсчета статистики.
- `OptionFilter.java` - реализует парсинг аргументов из командной строки с помощью Apache Commons CLI.

## Установка и сборка

1. Клонируйте репозиторий:
    ```bash
   git clone https://github.com/ShkNikn1ce/FilterFile.git

2. Сборка проекта с помощью Gradle:
    ``` bash
   gradle build
   
3. Убедитесь, что библиотека Apache Commons CLI добавлена в зависимости:
    ```implementation("commons-cli:commons-cli:1.6.0")```

---

## Запуск

java -jar build/libs/FilterFile-1.0-SNAPSHOT-all.jar -0 . -p sample- -f input_1.txt input_2.txt



## Параметры командной строки

- " -o ": Путь выходных файлов. Пример: -o ./output
- " -p ": Префикс для имен выходных файлов: -p sample- 
- " -s ": Вывести краткую статистику. Пример: -s
- " -f ": Вывести полную статистику. Пример: -f
